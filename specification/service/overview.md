# 서비스 개요: Coupang Review AI

## 서비스 한 줄 정의

쿠팡 셀러가 경쟁 상품 URL을 입력하면, AI가 리뷰를 분석해 **"이걸 고치면 경쟁사를 이긴다"** 는 인사이트를 제공하는 SaaS 도구.

---

## 타겟 사용자

- **주 타겟**: 쿠팡 스토어 운영 셀러 (월 매출 100만~5000만원 규모, 전업 또는 부업)
- **세부 타겟**:
  - 신규 상품 출시 전 시장 조사하는 셀러
  - 기존 상품의 판매 부진 원인을 찾는 셀러
  - 경쟁사보다 우위를 가져가고 싶은 셀러

---

## 핵심 사용자 흐름

1. 쿠팡 경쟁 상품 URL 입력 (최대 3개 동시 비교)
2. 리뷰 자동 수집 (최근 100~500개)
3. AI 분석 실행:
   - 반복 불만 패턴 추출 ("배송 포장이 허술하다" 37건)
   - 긍정 구매 이유 추출 ("가성비가 좋다" 52건)
   - 평점별 감성 분포
4. 셀러용 리포트 출력:
   - "고쳐야 할 것 TOP 3"
   - "경쟁사 약점 (내가 파고들 기회)"
   - "고객이 구매를 결정하는 핵심 요인"
5. 리포트 저장 / PDF 내보내기

---

## MVP 기능 (v1 범위)

- [ ] 쿠팡 상품 URL 입력 → 리뷰 수집
- [ ] AI 리뷰 분석 (불만 패턴, 긍정 요인, 개선 포인트)
- [ ] 경쟁사 vs 내 상품 비교 분석 (URL 2개 입력)
- [ ] 분석 리포트 화면 표시
- [ ] 분석 히스토리 저장 (회원가입/로그인)
- [ ] 리포트 PDF/이미지 내보내기

## v1 명시적 제외

- 스마트스토어, 네이버쇼핑 연동 (v2)
- 실시간 모니터링 알림 (v2)
- 팀/멀티 계정 (v2)
- 리뷰 조작 탐지 (v2)
- 모바일 앱 (v2)
